# cyberpunk.nvim
Lua port of the vim [cyberpunk colorscheme](https://github.com/thedenisnikulin/vim-cyberpunk)

### Installation

#### Packer

```vim
use 'samueljoli/cyberpunk.nvim'
```

#### Plug
```vim
Plug 'samueljoli/cyberpunk'
```

### Usage
```lua
require('cyberpunk').setup {
  theme = 'dark'
}
```
